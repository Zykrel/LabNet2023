Juego de adivinar el número

Adivine el numero secreto para ganar!!!
Hay una cantidad limitada de intentos pero si el puntaje llega a 0 se Pierde :(.

Ingresa un número entre 1 y 20:

¿Como se juega?

1- Se intenta adivinar el numero secreto para ganar.
2- Cada vez que se adivina el numero secreto puede continuar, pero cada intento se descuenta de su puntaje Actual.
3- Por defecto se tiene 20 intentos :D.
4- El puntaje se va sumando a medida que el numero es adivinado.
5- Al llegar a 0 se pierde, para volver a empezar ingresa un numero!.
6- Divertirse!!.
